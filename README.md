# Personal Website
> This website was built in Vue.js which includes a history of present and past clients.

## Technologies
> Project is created with:  
  * Vue.js 2.5
  * Axios 2.5
  * Firebase 5.5
  * Sass 2.5

## Setup

# install dependencies
npm install
yarn install

# serve with hot reload at localhost:8080
npm run dev
yarn start
